Most users tend to browse websites from top to bottom and left to right. Based on this pattern, I wanted to examine whether placing the upload and browsing functions at the top of the page would be more effective than placing them on the left-hand side. To test this, I moved the main UI elements from the left panel to the top of the interface, while keeping the metric cards on the left as a control condition. This layout allowed me to compare user behavior and evaluate which placement better encouraged data uploading and browsing.
